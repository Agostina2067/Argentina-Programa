Conceptos Abordados en la Primera Clase

Navegadores

Un navegador web es un software que nos permite el acceso a la Web, interpretando la información de distintos tipos de archivos, y mostrándola al internauta de manera adecuada.

Desarrollo Frontend

Es la parte del desarrollo web que se dedica a la parte frontal de un sitio web, del diseño de un sitio web, desde la estructura hasta los estilos como colores, fondos, tamaños, animaciones y efectos. Es todo lo que el visitante ve y experimenta de forma directa.

Lenguajes Utilizados:

  • HTML
  • CSS
  • JavaScript

Desarrollo Backend

Procesa la información que alimentará el Frontend de datos. Es la capa de acceso a datos, es la lógica tecnológica que hace que una página web funcione.
